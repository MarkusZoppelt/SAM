The CVSS v3.0 equations are defined below.<br /></p><p>8.1. Base<br /></p><p><br /></p><p>The Base Score is a function of the Impact and Exploitability sub score equations. Where the Base score is defined as,<br /></p><p><br /></p><p>If (Impact sub score <= 0) 0 else,<br /></p><p>Scope Unchanged[4] Round up (Minimum [(Impact + Exploitability), 10])<br /></p><p>Scope Changed Round up (Minimum [1.08 × (Impact + Exploitability), 10])<br /></p><p><br /></p><p>and the Impact sub score (ISC) is defined as,<br /></p><p><br /></p><p>Scope Unchanged 6.42 × ISCBase<br /></p><p>Scope Changed 7.52 × [ISCBase-0.029] - 3.25 × [ISCBase-0.02]15<br /></p><p><br /></p><p>Where,<br /></p><p><br /></p><p>ISCBase = 1 - [(1-ImpactConf) × (1-ImpactInteg) × (1-ImpactAvail)]<br /></p><p><br /></p><p>And the Exploitability sub score is,<br /></p><p><br /></p><p>8.22 × AttackVector × AttackComplexity × PrivilegeRequired × UserInteraction<br /></p><p>8.2. Temporal<br /></p><p><br /></p><p>The Temporal score is defined as,<br /></p><p><br /></p><p>Round up(BaseScore × ExploitCodeMaturity × RemediationLevel × ReportConfidence)<br /></p><p><br /></p><p>8.3. Environmental<br /></p><p><br /></p><p>The environmental score is defined as,<br /></p><p><br /></p><p>If (Modified Impact Sub score =< 0)  0 else,<br /></p><p>If Modified Scope Unchanged Round up(Round up (Minimum [<br /></p><p>          (M.Impact + M.Exploitability), 10])<br /></p><p>          × Exploit Code Maturity<br /></p><p>          × Remediation Level<br /></p><p>          × Report Confidence)<br /></p><p><br /></p><p>If Modified Scope Changed Round up(Round up (Minimum [1.08<br /></p><p>          × (M.Impact + M.Exploitability), 10])<br /></p><p>          × Exploit Code Maturity<br /></p><p>          × Remediation Level<br /></p><p>          × Report Confidence)<br /></p><p><br /></p><p>And the modified Impact sub score is defined as,<br /></p><p><br /></p><p>If Modified Scope Unchanged 6.42 × [ISCModified]<br /></p><p>If Modified Scope Changed 7.52 × [ISCModified-0.029] - 3.25 × [ISCModified-0.02]15<br /></p><p><br /></p><p>Where,<br /></p><p><br /></p><p>ISCModified = Minimum[[1-(1-M.IConf × CR)×(1-M.IInteg × IR)×(1-M.IAvail × AR)],0.915]<br /></p><p><br /></p><p>The Modified Exploitability sub score is,<br /></p><p><br /></p><p>8.22 × M.AttackVector × M.AttackComplexity × M.PrivilegeRequired × M.UserInteraction <br /></p>
